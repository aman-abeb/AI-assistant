A WordPress plugin to generate AI-powered content using OpenAI's GPT API.

== Description ==
This plugin allows you to generate content using OpenAI's GPT API directly from the WordPress post editor.

== Installation ==
1. Upload the `ai-content-assistant` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the plugin settings page and enter your OpenAI API key.
4. Start generating content in the post editor.
